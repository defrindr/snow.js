# Snow.JS
library untuk menampilkan salju menggunakan javascript


## Cara menggunakan
Dapat dengan mendownload repo ini atau
import script dari CDN
```
<html>
  <head>
    <title>snow.js</title>
  </head>
  <body>
    ...
    <script src="https://cdn.jsdelivr.net/gh/defrindr/snow.js@1.0.0/src/snow.min.js"></script>
  </body>
</html>
```
